## Key Decisions & Lessons
- **Tailwind v4 Sizing Configuration:** Custom font sizes defined under `:root` are not registered as class utilities by default in Tailwind v4; they must be declared in `@theme` using the `--font-size-` prefix.
- **RTL Typography Contrast:** Contrast must be tested with active variable bindings to ensure fallback styles do not inherit incorrect dark colors.
- **Background Palette Priority:** Stick to the dark navy blue (`#091521`) as the main background layer for dark sections, treating pure black only as a minor shadow helper.
